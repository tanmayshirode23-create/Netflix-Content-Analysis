# 🎬 Netflix Content Analysis Dashboard

## 📊 Project Overview

This project analyzes Netflix content data to uncover insights about content distribution, trends, and audience targeting.

## 🛠 Tools Used

* Python (Pandas, NumPy)
* Power BI
* DAX

## 📁 Dataset

Netflix Movies and TV Shows dataset from Kaggle.

## 🔍 Key Insights

* Majority of content is Movies compared to TV Shows
* Significant growth in content after 2015
* Drama and International Movies dominate the platform
* Most content is rated TV-MA and TV-14

## 📈 Dashboard Features

* Interactive filters (Country, Type, Year)
* Dynamic title using DAX
* KPI cards for quick insights
* Visualizations for trends, genres, ratings, and countries
